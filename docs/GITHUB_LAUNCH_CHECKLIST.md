# GitHub launch checklist

## Before publishing

- [ ] Create a new public GitHub repository named `codex-platformio-agent-kit`
- [ ] Push this project to `main`
- [ ] Confirm GitHub Actions runs
- [ ] Add a short repository description:
      `Codex-ready PlatformIO workflow kit for ESP32 and Raspberry Pi Pico projects`
- [ ] Add topics:
      `codex`, `platformio`, `esp32`, `raspberry-pi-pico`, `embedded`, `agents`, `firmware`, `arduino`
- [ ] Create the first release: `v0.1.0`
- [ ] Pin the repository on your GitHub profile
- [ ] Share it with a few embedded/AI developer communities and ask for feedback

## README proof points to add after launch

- CI badge
- Star count, if meaningful
- Screenshots of CI passing
- Example Codex session transcript, if safe to share
